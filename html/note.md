# Note

## HTML

### 基础标签

- html : 根标签

- head : 包含了文档的元信息，比如标题、描述、关键词、编码、样式表、脚本等。

- body : 包含了文档的主要内容，比如文本、图片、视频、音频、表格、表单等。

- h1~h6 : 六级标题标签，用于定义不同级别的标题。

- p : 段落标签

- br/ : 换行标签他是一个单标签

- hr/ : 水平线标签，用于分割内容。

- strong : 加粗标签

- em : 斜体标签

- u : 下划线标签

- strike : 删除线标签

- img : 图片标签，用于插入图片。
    ```html
    <img src="image.jpg" alt="图片描述" width="100" height="100" border=10px />
    ```

    - alt : 只有当图片没有加载时, 会显示alt属性中的内容
    - width/height : 设置图片的宽度和高度
    - border : 设置图片的边框


- a : 超链接标签，用于创建超链接。
    ```html
    <a href="https://www.baidu.com">百度</a>
    ```

    - href : 链接地址

    - target : 链接打开方式，"_blank"表示在新窗口打开

- table : 表格标签，用于创建表格。

    - thead : 用于表示表头 
    - tr : 表格行标签，用于创建表格的行。
    - td : 表格单元格标签，用于创建表格的单元格。
    - th : 表格标题标签，用于创建表格的标题。
    - cellspacing : 设置单元格之间的间距。
    - cellpadding : 设置单元格内容与单元格边框之间的间距。
    - border : 设置单元格边框的宽度。
    - width/height : 设置表格的宽度和高度。
    - rowspan/colspan : 设置单元格的合并。

- 列表标签
    - ul : 无序列表标签，用于创建无序列表。
    - ol : 有序列表标签，用于创建有序列表。
    - li : 列表项标签，用于创建列表项。
    - dl : 定义列表标签，用于创建定义列表。
    - dt : 定义标题标签，用于创建定义列表的标题。
    - dd : 定义描述标签，用于创建定义列表的描述。
